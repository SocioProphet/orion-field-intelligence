# Source handling policy

This directory stores upstream reference materials used to inform the repo-native contracts, ontology modules, examples, and docs.

## Rules
- Repo-native schemas, ontology, and docs are the source of truth.
- Source documents are references, not product artifacts.
- Redistribution and licensing status for third-party source PDFs must be reviewed before any public release.
- If redistribution is not clearly allowed, replace in-repo binaries with citations and our synthesis.

## Current status
- COIN source status: deduped-identical
- Public release posture: keep repository private until source redistribution is reviewed.
