# Guardrails (COIN / Stabilization)

## Default posture
- Aggregate, identity-neutral indicators by default.
- Human review for interventions; audit trail for rationale and outcomes.
- Explicit uncertainty: staleness, confidence, missingness, contradictions.

## Anti-Goodhart measures
- Red-team metrics for gaming or perverse incentives.
- Track second-order effects (e.g., service restored but intimidation increased).
- Require measurement plans and post-action evaluation.

## Access and disclosure
- Role-based access controls for sensitive sources.
- Redaction policies for coalition/partner visibility differences.
