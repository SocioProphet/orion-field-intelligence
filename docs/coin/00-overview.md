# COIN-Optimized Stabilization Loop (Orion / OFIF)

This module integrates counterinsurgency (COIN) / stabilization doctrine into Orion’s field-intelligence loop as an identity-neutral, effects-measured feedback system.

## Core claim
Operational success is determined by legitimacy and population confidence in governance, security, services, and dispute resolution. The “battlespace” includes the population and its information environment, not only terrain and opposing forces.

## Operational loop
Observe → Diagnose (legitimacy strain + grievances + influence shifts) → Intervene (multi-channel) → Measure effects → Adapt.

## Outputs
Orion emits Decision Cards that propose next-best-actions with:
- time bounds and refresh cadence
- explicit assumptions and constraints
- fog-of-war status (staleness, coverage gaps)
- expected effects on legitimacy indicators
- a measurement plan to confirm or falsify assumptions

## Guardrails
This module defaults to aggregate and identity-neutral indicators, with explicit uncertainty and human review requirements. It is intended for transparent stabilization workflows, not individual-level targeting.
